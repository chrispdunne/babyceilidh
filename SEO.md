## 1) Improve homepage title tag

- Update the homepage `<title>` to include intent + location.
- Suggested title:
  - `Family Ceilidh Dancing in Sussex | Little Folk Ceilidh`

## 2) Add an FAQ section on the homepage

- Add a short FAQ block to capture long-tail search intent.
- Suggested questions:
  - What age is Little Folk Ceilidh for?
  - Do we need dance experience?
  - Where in Sussex are events held?
  - How long does a session last?
  - Can we bring babies and siblings?

## 3) Create separate pages for each event or venue

- Add a dedicated URL per event (or per venue) instead of only one page.
- Include:
  - Event date and time
  - Venue location details
  - Ticket link and availability
  - Travel/parking basics where relevant

## 4) Keep Event schema up to date

- Ensure JSON-LD `Event` data always matches live event details.
- Keep these fields current:
  - `startDate`
  - `location`
  - `offers.url`
  - `offers.availability`
  - `eventStatus`

## 5) Expand local relevance in copy

- Add natural location terms where accurate (for example: Shoreham, Brighton, Hove, West Sussex).
- Mention nearby areas families might search from.
- Keep wording human-first (avoid keyword stuffing).
